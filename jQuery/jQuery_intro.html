jQuery是一个JavaScript函数库，该函数库可以简化web应用的开发过程中HTML元素遍历，事件处理，动画以及Ajax交互等操作。如同jQuery网站首页所说，jQuery可以使你花费更少的时间完成更多的工作。

jQuery库包含以下特性：[1]

  • HTML元素选取
  • HTML元素操作
  • CSS操作
  • HTML事件函数
  • JavaScript特效和动画
  • HTML DOM遍历和修改
  • Ajax
  • Utilities

下载与安装

方法1 CDN

由于jQuery的广泛使用，Google,Microsoft等网站都提供了jQuery的CDN，可以在script标签的src中直接填入地址即可。好处是充分利用带宽，减少本站耗费的流量。

  • Google: https://ajax.googleapis.com/ajax/libs/jquery/1.7.2/jquery.min.js
  • Microsoft: http://ajax.aspnetcdn.com/ajax/jQuery/jquery-1.7.2.min.js
  • jQuery: http://code.jquery.com/jquery-1.7.2.min.js

方法2 下载

当然，本地开发的话，有个副本还是比较方便的。最新的jQuery下载地址，分为压缩版和未压缩版，压缩版体积小，适合用在生产环境中，未压缩的版本可以平时学习所用。

方法3 Git

jQuery是一个开源项目，代码托管在Github，想要获得最新的测试版本可以在Github上下载回来，关注该项目，甚至可以为该项目贡献源码。不过，测试版本不要用在实际生产环境中。

Hello jQuery

下面我们简单地写个测试页面当作范例。当点击页面中的超级链接的时候，弹出“Hello world”对话框。 

<html>                                                                  
<head>                                                                  
	<script type="text/javascript" src="jquery.js"></script>          
	<script type="text/javascript">                                         
	      // JavaScript代码添加在此处                                    
	</script>                                                               
</head>                                                                 
<body>                                                                  
	<!-- 文本内容添加在此处 -->                                        
</body>                                                                 
</html>

记得把jQuery的脚本地址更换成自己的地址。在<body>标签中加入超级链接标签： 

<a href="">点我</a>

然后在JavaScript代码添加处加入以下代码： 

$(document).ready(function(){
	$("a").click(function(){
		alert("Hello world!");
		});
	});

保存，刷新页面，点击“点我”，可以看到弹出了"hello .html
