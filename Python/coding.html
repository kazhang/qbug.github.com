Python 字符编码

注明编码

Python按照系统默认的字符编码，一般为ASCII,若代码中出现中文字符等非ASCII字符，编译则会出错。所以应当在代码开头注明所用编码 

#coding=utf-8

转码

Python提供了encode和decode两个函数对字符串进行编码和解码，如下边这个例子将UTF-8转换为GB2312： 

str1="中文字符"
str2=str1.decode('utf-8').encode('gb2312')

字符串长度

如果直接对含中文字符的字符串串进行len操作，得到的不是正确的长度（貌似Python用3个\x__类型字符代替一个中文字符），而应先对其解码，再求其长度。 

str="中文字符"
print len(str)
str=str.decode('utf-8')
print len(str)
