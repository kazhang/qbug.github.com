C++ Primer 笔记

  1. 非const变量默认为extern。要使const变量能够在其他的文件中访问,必须地指定它为extern(注：其他文件均需要声明使用的变量为extern)。
  2. 用 class 和 struct 关键字定义类的唯一差别在于默认访问级别:默认情况下,struct 的成员为 public,而 class 的成员为 private。
  3. typedef不是简单的文本扩展，比如： 
    		string str="123";
		typedef string *pstring;
		const pstring cstr=&str;//实际上等价于string *const cstr;必须赋初值。
		pstring const cstr2=&str;//与上面一句等价
  4. 异常处理，参照：标准异常类列表 
    	  try{
		//code
		throw runtime_error("exception description");
		//code
	  }catch(runtime_error err){
		cout<<err.what()<<endl;
	  }
  5. 下标操作符比圆括号优先级高：foo(int (&arr)[10])；*操作符也一样：int (*martix)[10]
  6. 在无法列举出传递给函数的所有实参的类型和数目时，可以使用省略符形参。省略符形参有下列两种形式： 
    		  void foo(parm_list,...);
		  void foo(...);
  7. 仅当形参是引用或指针时,形参是否为 const 才有影响,但不能基于指针本身是否为const来实现函数的重载。
